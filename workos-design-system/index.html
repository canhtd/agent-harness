WorkOS Design System

The product design language for the WorkOS agent workspace, derived from Linear's live app: color in LCH, the Inter Variable type scale, 28px controls, layered popover shadows, and the full status & priority icon set. One quiet indigo, content over chrome.

Accent #5E6AD2 Inter Variable 28px controls 8px radius

Color

Backgrounds and text resolve through a four-step neutral scale tinted with a faint blue hue (282°). The interface stays near-monochrome so status color carries all the signal.

Controls share a 28px height. Radii step from 6px editor blocks to a full pill, with 8px as the workhorse control radius.

Elevation

The system avoids heavy shadows. Surfaces separate by a faint background step or a 0.5px border; only floating layers cast the three-stop popover shadow.

Status icons

The workflow set: a circle that fills clockwise as work progresses, color-coded by state. 14×14, exported as individual .svg in icons/status/.

Priority icons

Bar-chart marks that rise with priority, plus the filled square for Urgent. 16×16, in icons/priority/.

Buttons

28px tall, 500 weight, 13px. The primary indigo carries one action per view; the empty-state CTA goes pill. Keyboard hints sit inline.

Component states

Mỗi control có 5 state. Press dùng scale(0.96) ngắt được; focus-visible là vòng #6D78D5; disabled hạ opacity .5 và khoá con trỏ.
